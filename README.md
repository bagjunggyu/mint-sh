### Linux Mint 22.x 설치 후 개인 설정 자동화 작업 ###
##### 맥북으로 설치하는 경우에는 bcmwl-kernel 소스 설치 및 fusuma 설치(#제거)
##### 선호하는 clementine, moc, gparted, boot-repair, ranger, tilda, mpv를 설치하고
##### 잘 사용하지 않는 rhythmbox, celluloid,  thunderbird를 제거합니다.
##### vsftpd, ssh를 설치하고 미리 작업해 둔 내용으로 설정합니다.
##### 한글 입력기 kime를 설치하고 미리 설정한 값으로 지정합니다.
##### mint-sh.tar.gz 파일을 다운로드해서 mint.sh 파일을 mint-sh 폴더에서 실행하면 됩니다 
##### sudo sh mint.sh
